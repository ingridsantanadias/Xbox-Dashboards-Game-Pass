# 🎮 Xbox Subscription Dashboard Analysis

Este projeto consiste na análise e estruturação de dados de assinantes do ecossistema Xbox (Core, Standard e Ultimate). O objetivo é fornecer insights sobre receita, comportamento de renovação e adesão a passes de temporada (EA Play e Minecraft), utilizando uma identidade visual fiel à marca.

## 📊 Sobre os Dados

Os dados brutos estão consolidados no arquivo `xbox-dashboard.xlsx` e abrangem o período de Janeiro a Dezembro de 2024.

### 🗂️ Estrutura das Bases
- **Subscriber ID:** Identificação única do cliente.
- **Plan:** Nível da assinatura (Core, Standard, Ultimate).
- **Subscription Type:** Periodicidade (Mensal, Trimestral, Anual).
- **Add-ons:** Status de aquisição do *EA Play Season Pass* e *Minecraft Season Pass*.
- **Financials:** Cálculo de valor total considerando preço base, passes extras e cupons de desconto aplicados.
